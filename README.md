# 🩺 Diabetes Prediction System using Machine Learning

## 📌 Project Overview
The **Diabetes Prediction System** is a Machine Learning based healthcare project developed using **Python** and **Scikit-learn**.  
This system predicts whether a person is likely to have diabetes based on medical parameters from the **PIMA Indians Diabetes Dataset**.

The project focuses on data preprocessing, feature scaling, model training, and prediction analysis using different Machine Learning algorithms.

---

## 🎯 Objectives
- To analyze diabetes-related medical data
- To build an accurate prediction model
- To compare different Machine Learning techniques
- To improve prediction accuracy using preprocessing and optimization

---

## 🚀 Key Features
✔ Diabetes risk prediction using Machine Learning  
✔ Data preprocessing and feature scaling  
✔ Model training and evaluation  
✔ Gradient Boosting implementation  
✔ Trained model storage using Pickle (`.pkl`)  
✔ Jupyter Notebook based experimentation

## 📂 Project Structure

| File Name | Description |
|-----------|-------------|
| `diabetes_model.pkl` | Saved trained Machine Learning model |
| `final_model.pkl` | Final optimized prediction model |
| `pima-indians-diabetes.csv` | Original diabetes dataset |
| `scaled_diabetes_dataset.csv` | Preprocessed and scaled dataset |
| `trial and error.ipynb` | Notebook containing model experimentation |
